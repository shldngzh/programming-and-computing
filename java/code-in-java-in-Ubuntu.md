# Code in Java on Ubuntu

## install JRE and JDK
```bash
$ sudo apt-get update
$ sudo apt-get install default-jre
$ sudo apt-get install default-jdk
$ sudo apt-get update
```

# compile and run
```bash
$ javac <file-name>.java // eg. HelloWorld.java
$ java <new-compiled-file>.class // eg. HelloWorld.class
```
